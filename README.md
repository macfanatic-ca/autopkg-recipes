# AutoPkg Recipes

## Function
Used to deploy the latest software with ease

## Tested Environment(s)
* OS X 10.11 & macOS 10.12

## Dependancies
1. autopkg
2. git

## Configuration
1. `autopkg repo-add https://gitlab.com/ClassmateTeam/autopkg-recipes`
